# Linux System Monitor

Ferramenta em Python para monitoramento de recursos do sistema Linux.

## Funcionalidades

- Monitoramento de CPU
- Monitoramento de memória
- Uso de disco
- Informações do sistema em tempo real

## Tecnologias utilizadas

- Python
- Linux
- Biblioteca psutil

## Como executar

1. Clone o repositório

git clone https://github.com/seuusuario/linux-system-monitor.git

2. Entre na pasta

cd linux-system-monitor

3. Execute

python main.py
